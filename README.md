## Musics é um projeto Full Stack desenvolvido no último módulo da Kenzie Academy.

No Back end foi utilizado NestJS com TypeScript, com uma documentação estruturada sobre as rotas e um arquivo com um insomnia contendo todas as requisições. Basta clonar o repositóriol e instalar as dependências dentro da pasta back-end com os comandos:

	-npm i
	-npm run start:dev
	
Então poderá executar todas as rotas!

Ele está conectado com uma nuvem gratuita para poder subir novas músicas dentro da aplicação! possui um sistema de login e autenticação de usuário. Além disso, está conectado com um banco de dados no DBeaver. Irei deixar um link para um vídeo demonstrando a aplicação funcionando!



Já no Front-End foi utilizado React com Tail-Wind, e para acessar também precisará instalar as dependências com o comando

	-npm i 
	-npm run dev
	
Importante: cada um deve ser feito em um terminal diferente para poder rodar ambos localmente.



## INSOMNIA

O que é?
É um aplicativo de código aberto utilizado para testar APIs (interface de programação de aplicações)

para baixar: https://insomnia.rest/download

Então basta importar o arquivo insomnia_routes dentro do programa, e poderá verificar todas as rotas funcionando :)


## DBeaver

O que é?
É um aplicativo de código aberto com uma interface para poder verificar o banco de dados de uma forma mais fácil.

para baixar: https://dbeaver.io/download/

Neste projeto, estarei usando o PostgreSQL, portanto precisará criar um banco de dados na variável de ambiente .env!
